val reader : ?read_body_len:int -> boundary -> input -> reader

reader ?read_body_len boundary input creates reader. The default value for read_body_len is 1KB.

val parse_part : reader -> read_result promise

parse_part ?read_body_len ~boundary reader parse http multipart body and returns a read_result.

read_body_len determines the size of the multipart body to read in bytes. By default 1KB.
